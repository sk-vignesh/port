# Plan 02-02: Rewire QuickTradeButton FAB + Dashboard Entry Point

## Objective
Wire the `UnifiedTradeSearch` modal into the existing `QuickTradeButton` FAB (replacing the static `/portfolios?action=buy` link). Add a prominent "Search & Trade" entry point on the Dashboard for first-use discovery. When a user taps "Buy" or "Quick Trade", they get the live search → portfolio picker → trade form flow.

## Context
`QuickTradeButton.tsx` is a fixed FAB in the bottom-right corner, used globally. Currently "Buy" and "Sell" link to static portfolio pages with no search integration. This plan wires the `UnifiedTradeSearch` modal into this button. The modal was built in Plan 02-01.

### Task 2: Add Search-to-Trade Hero Entry on Dashboard

<read_first>
- `app/(app)/page.tsx` (lines 100-160 — the hero section with net worth)
- `components/UnifiedTradeSearch.tsx` (props: `open`, `onClose`)

<action>
In the Dashboard (`app/(app)/page.tsx`), add a "Record a Trade" call-to-action in the hero section below the net worth figure.

Find the section that renders the overall net worth and add below it:

```tsx
{/* Search-to-trade CTA */}
<DashboardTradeButton />
```

Create a small client component `components/DashboardTradeButton.tsx`:
```tsx
'use client'
import { useState } from 'react'
import UnifiedTradeSearch from './UnifiedTradeSearch'

export default function DashboardTradeButton() {
  const [open, setOpen] = useState(false)
  return (
    <>
      <button
        onClick={() => setOpen(true)}
        style={{
          display: 'inline-flex', alignItems: 'center', gap: 8,
          padding: '10px 20px', borderRadius: 'var(--radius-md)',
          background: 'linear-gradient(135deg, #6366f1, #8b5cf6)',
          border: 'none', color: '#fff', fontWeight: 700, fontSize: '0.875rem',
          cursor: 'pointer', boxShadow: '0 4px 16px rgba(99,102,241,0.35)',
          transition: 'all 0.2s',
        }}
      >
        ↑↓ Record a Trade
      </button>
      <UnifiedTradeSearch open={open} onClose={() => setOpen(false)} />
    </>
  )
}
```

This gives users a second, more prominent entry point on the most-visited page.
</action>
