# Plan 03-01: DB Migration + AMFI NAV Edge Function

## Objective
Add `MUTUAL_FUND` as a new asset class in the system. Create a Supabase migration that adds a `folio_number` text column to `portfolio_transactions` and adds MUTUAL_FUND to the metadata. Create a new `amfi-nav` Edge Function that proxies the **AMFI India open data API** to fetch daily NAV for any mutual fund scheme by code or name search.

## Context
The AMFI (Association of Mutual Funds in India) provides a **completely free, no-auth API**:
- Search by name: `https://api.mfapi.in/mf/search?q=<name>`
- Get historic NAV: `https://api.mfapi.in/mf/<schemeCode>`
- Latest NAV: `https://api.mfapi.in/mf/<schemeCode>/latest`

The pattern follows `price-search` exactly: Edge Function proxies external API to avoid CORS. No auth required (returns public NAV data only).

Mutual Funds use **units** (not shares) and **NAV** (not price per share). The schema already supports this via the `unit_type` column added in migration 009 — just set `unit_type = 'UNITS'`. The existing `shares` column stores units × 100_000_000 and `amount` stores invested amount × 100.

  ADD COLUMN IF NOT EXISTS amfi_scheme_code TEXT;

-- Index for NAV fetching by scheme code

  ON securities (amfi_scheme_code)
  WHERE amfi_scheme_code IS NOT NULL;

COMMENT ON COLUMN portfolio_transactions.folio_number IS 'Mutual fund folio number, e.g. 12345678/12';
COMMENT ON COLUMN securities.amfi_scheme_code IS 'AMFI scheme code for NAV fetch, e.g. 120503';

### Task 3: Create amfi-nav Edge Function

<action>
Create `supabase/functions/amfi-nav/index.ts`:

```ts
/**
 * Supabase Edge Function: amfi-nav
 *
 * GET /functions/v1/amfi-nav?q=<name>           — search by name
 * GET /functions/v1/amfi-nav?code=<schemeCode>  — get latest NAV by scheme code
 *
 * Proxies AMFI India free open data API:
 *   https://api.mfapi.in
 *
 * No auth required (public NAV data only).
 */
import { handleCors, json } from '../_shared/cors.ts'

Deno.serve(async (req: Request) => {
  const corsResp = handleCors(req)
  if (corsResp) return corsResp

  const url    = new URL(req.url)
  const q      = url.searchParams.get('q')?.trim()
  const code   = url.searchParams.get('code')?.trim()

  try {
    // Search by name
    if (q && q.length >= 2) {
      const res = await fetch(
        `https://api.mfapi.in/mf/search?q=${encodeURIComponent(q)}`,
        { signal: AbortSignal.timeout(6000) }
      )
      if (!res.ok) return json([])
      const schemes: Array<{ schemeCode: number; schemeName: string }> = await res.json()
      return json(
        schemes.slice(0, 12).map(s => ({
          code:     String(s.schemeCode),
          name:     s.schemeName,
          currency: 'INR',
          type:     'MUTUALFUND',
        }))
      )
    }

    // Get NAV by scheme code
    if (code) {
      const res = await fetch(
        `https://api.mfapi.in/mf/${encodeURIComponent(code)}/latest`,
        { signal: AbortSignal.timeout(6000) }
      )
      if (!res.ok) return json(null)
      const data: { status: string; data: Array<{ date: string; nav: string }> } = await res.json()
      const latest = data?.data?.[0]
      if (!latest) return json(null)
      return json({
        code,
        date: latest.date,
        nav:  parseFloat(latest.nav),  // in INR, e.g. 234.56
      })
    }

    return json({ error: 'q or code parameter required' }, 400)
  } catch {
    return json([])
  }
})
```
</action>
