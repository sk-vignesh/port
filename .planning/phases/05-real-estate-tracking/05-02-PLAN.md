# Plan 05-02: PropertyCard Portfolio Wiring + Rental Income Aggregation

## Objective
Wire `PropertyCard` into the portfolio detail page so REAL_ESTATE portfolios show a property summary card section above the transaction grid — similar to how FDCard works for FIXED_INCOME (added in Phase 4). Aggregate rental income and stamp duty from transaction data to power the property valuation metrics.

## Context
The portfolio detail page (`app/(app)/portfolios/[id]/page.tsx`) was updated in Phase 4 to:
- Include `FDCard` import
- Read `assetClass` from portfolio
- Filter BUY txns with `face_value` for FD display

We follow the same pattern for `REAL_ESTATE`. The `security_prices` table holds user-entered current valuations for properties (via `/securities/{id}` — the manual price update UI).

### Data aggregation needed per property

For each unique `security_id` in REAL_ESTATE BUY transactions:
```
purchaseCost  = sum(amount) WHERE type = 'BUY'
annualRental  = sum(amount) WHERE type = 'INTEREST' AND year = current_year
area          = first(area) WHERE type = 'BUY'
currentValue  = latest security_prices entry OR null
stampDuty     = from portfolio_transaction_units WHERE type = 'FEE' (joined separately)
```

```ts
// Aggregate real estate properties from REAL_ESTATE portfolios
type AggregatedProperty = {
  securityId:   string
  name:         string
  purchaseCost: number   // BIGINT×100
  stampDuty:    number   // BIGINT×100 — NOTE: requires fetching portfolio_transaction_units
  area:         number | null
  annualRental: number   // BIGINT×100
  currentValue: number | null  // from security_prices map (not available here yet — null)
}

const reTxns: AggregatedProperty[] = assetClass === 'REAL_ESTATE'
  ? (() => {
      const TODAY_YEAR = new Date().getFullYear()
      const map = new Map<string, AggregatedProperty>()

      for (const tx of transactions ?? []) {
        const rawTx = tx as unknown as {
          area?: number | null
          securities?: { id?: string; name?: string } | null
        }
        const secId  = rawTx.securities?.id  ?? tx.security_id ?? 'unknown'
        const secName = rawTx.securities?.name ?? 'Property'

        if (!map.has(secId)) {
          map.set(secId, {
            securityId: secId, name: secName,
            purchaseCost: 0, stampDuty: 0,
            area: null, annualRental: 0, currentValue: null,
          })
        }
        const p = map.get(secId)!
        if (tx.type === 'BUY') {
          p.purchaseCost += tx.amount ?? 0
          if (!p.area && rawTx.area) p.area = rawTx.area
        }
        if (tx.type === 'INTEREST') {
          const txYear = new Date(tx.date).getFullYear()
          if (txYear === TODAY_YEAR) p.annualRental += tx.amount ?? 0
        }
      }
      return [...map.values()].filter(p => p.purchaseCost > 0)
    })()
  : []
```
