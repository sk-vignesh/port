# Plan 08-02: CAS PDF Parser

## Objective
Allow users to upload their CAMS/KFintech/CDSL Consolidated Account Statement (CAS) PDF and auto-import all mutual fund holdings into the app.

## Background on CAS PDFs
- CAMS and KFintech are the two main MF RTAs (Registrar & Transfer Agents).
- CAS PDFs are generated at mfcentral.com or sent via email from CAMS/KFintech.
- The PDF contains: investor name, PAN, folio numbers, scheme names, units held, NAV, current value.
- CAS PDFs are text-based (not scanned images), so they can be parsed with a PDF text extraction library.
- They are often password-protected â€” password is PAN (capitalized) + date of birth (DDMMYYYY).

## Implementation Strategy

### Step 1: Extract text from PDF (Deno Edge Function)
Use `pdf-parse` equivalent for Deno â€” `pdf.js` or `pdfjs-dist` via CDN import in Deno.

Alternative: use the `pdf-text-extract` approach with Supabase Edge Function receiving base64-encoded PDF.

**Recommended approach for Deno:**
```ts
import { getDocument } from 'https://cdn.jsdelivr.net/npm/pdfjs-dist@4.0.379/+esm'
```

### Step 2: Parse the extracted text
CAS PDF text follows a predictable format:
```
Folio No: 1234567890
Scheme: SBI Blue Chip Fund - Regular Plan (ISIN: INF200K01CN2)
Units: 1,234.567
NAV: â‚ą78.23
Value: â‚ą96,581.24

Transactions:
15-Jan-2024  SIP Purchase  â‚ą5,000.00  Units: 63.916
```

Use regex to extract:
- Folio number per fund house
- Scheme name (map to AMFI scheme code via AMFI master data)
- Units held
- NAV
- All transactions with dates and amounts

### Step 3: Create securities and transactions
For each MF scheme found:
1. Look up AMFI scheme code from scheme name (fuzzy match against AMFI master list)
2. `upsert` into `securities` (user_id, name, amfi_scheme_code, currency_code='INR')
3. Insert each transaction into `portfolio_transactions`:
   - type = `BUY` for purchases, `SELL` for redemptions
   - portfolio_id = user's `MUTUAL_FUND` portfolio (or `EQUITY` if no MF portfolio)
   - shares = units Ă— 100_000_000 (standard scaling)
   - amount = amount Ă— 100

### Step 4: Return summary
```json
{
  "funds": 12,
  "transactions": 87,
  "totalValue": 1250000,
  "errors": ["Could not match scheme: SBI Nifty 50 ETF"]
}
```
